# Docker - Python3 - MySQL 開発環境

## ファイル構成

```
├── README.md # このファイル
├── docker
│   └── Dockerfile # サーバ設定ファイル
├── docker-compose.yml # MySQL、phpMyAdminと一緒にpythonのサーバを起動するファイル
├── requirements.txt # pythonのパッケージ
└── src
    ├── cgi-bin
    │   └── cgipost.py
    ├── cgiserver.py
    └── index.html
```
